# Donnees Ouvertes Montreal (donnees-montreal-ca)

Donnees Ouvertes Montreal is a municipal government open-data portal for Canada running **CKAN**, exposing a standardized catalog API over ~401 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/donnees-montreal-ca/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=donnees-montreal-ca-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.11.5

## API
- **Donnees Ouvertes Montreal CKAN Action API** — base `https://donnees.montreal.ca/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://donnees.montreal.ca)
- Live endpoint verified: `https://donnees.montreal.ca/api/3/action/status_show` · ~401 datasets
